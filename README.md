# 番茄鐘

> 因為小朋友會使用3C，所以想要用番茄鐘的概念，設計一個計時器，每天提供小朋友一個固定時間，好比一小時給她自由使用，使用之前必須按下計時鐘，每Ｎ分鐘會響聲，提醒應該要休息，總時間超過一定值之後，也會顯示不應該再使用，家長可以登入觀察使用的狀況。
